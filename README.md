# Full Stack Expense Tracker

A full-stack expense tracking application built with React, Node.js, and SQLite.

## Features

- User authentication (register & login)
- Add, edit, and delete expenses
- JWT-based authentication
- Persistent login
- Protected routes

## Tech Stack

**Frontend**

- React
- Vite

**Backend**

- Node.js
- Express
- SQLite
- JWT Authentication

## Installation

### 1. Clone the repo

### 2. Install Backend

cd Backend
npm install

### 3. Install Frontend

cd ../Frontend/expense-tracker
npm install

### 4. Run App

Backend:
npm start
Frontend:
npm run dev
